# HOST-UI

Mobile Priority Component Library Based on React.

<img src="https://images-manager.oss-cn-shanghai.aliyuncs.com/2019/Screen%20Shot%202019-07-31%20at%204.49.59%20PM.png" width="100%" />

```bash
npm install host-ui
```

## Usage

```jsx
import 'host-ui/lib/hostui.css';
import { Button } from 'host-ui';
```

## Develop

```bash
npm start
```
